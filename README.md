# Data Cleaning & Structural Validation

## Overview
This project focuses on cleaning raw and unstructured data by handling duplicate entries, missing values, inconsistent formatting, incorrect data types, and invalid values. The goal is to transform the dataset into a clean and analysis-ready format while preserving data integrity.

## Tools Used
- Python
- Pandas
- NumPy
- Google Colab

## Steps Performed
- Loaded the raw CSV dataset
- Inspected rows, columns, missing values, and duplicates
- Removed duplicate rows
- Cleaned and standardized column names
- Standardized text values
- Handled missing values
- Corrected data types
- Validated the final cleaned dataset
- Saved the cleaned dataset as `cleaned_data.csv`

## Files in this Repository
- `Task1_Data_Cleaning.ipynb` → Colab notebook containing the complete code
- `cleaned_data.csv` → Final cleaned dataset
- `README.md` → Project documentation

## Outcome
The dataset was successfully cleaned and prepared for further analysis.
